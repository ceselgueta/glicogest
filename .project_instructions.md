# GlicoGest - Controle Glicêmico Gestacional

## Overview
App de controle de diabetes gestacional em português do Brasil. Design rosa/pink gradiente.

## Business Model (Updated)
- **Teste Grátis**: 4 dias, 1 PDF, uso único por conta
- **Mensal**: R$14,90 / 30 dias
- **3 Meses**: R$34,90 / 90 dias  
- **Gestação Completa**: R$59,90 / 270 dias (destacado como "Mais recomendado")
- Sem assinatura obrigatória - pagamento único por período
- Pagamento online NÃO integrado ainda (estrutura pronta para Stripe/Mercado Pago)
- Admin manual activation via POST /api/plans/admin-activate

## Plan System
- User model fields: plan, planExpiresAt, planStartedAt, hasUsedTrial, pdfReportsGenerated, paymentStatus
- Plan values: "free" (default/no plan), "free_trial", "monthly", "quarterly", "gestation_full", "premium" (legacy)
- computePlanStatus() in lib/plans.ts handles all plan logic
- JWT/session propagates all plan fields
- API routes check canRegisterReadings/canGeneratePdf from computePlanStatus()
- /planos page shows all 4 plans
- Dashboard shows PlanBanner with status/days remaining
- Expired users: can view dashboard/data but cannot register readings or generate PDFs

## Access Rules
- No plan: show CTA to choose plan (or trial if not used)
- Trial active: full access, 1 PDF limit
- Trial expired: view only, redirect to plans
- Paid active: full access, unlimited PDFs
- Paid expired: view only, redirect to plans

## Technical
- NextAuth with Credentials + Google SSO (Google keys still placeholder)
- Prisma with PostgreSQL
- Pages: / (landing), /login, /signup, /dashboard, /planos
- API routes: /api/readings, /api/readings/batch, /api/stats, /api/patient, /api/generate-pdf, /api/plans/status, /api/plans/activate-trial, /api/plans/admin-activate

## Design
- Pink/rose gradient theme
- Framer Motion animations
- Mobile-first responsive
- All text in Portuguese BR
